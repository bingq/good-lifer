## Voice Input Integration

Voice input is available in these contexts:

1. **Achievement Name** (Quick Log)
2. **Project Name** (New Project)
3. **Achievement Details/Notes** (Optional)
4. **Project Details** (Optional)
5. **Tree Hole Entry** (Primary input method)

**Interaction Pattern**: Hold-to-Speak
- User **holds** microphone button
- Visual feedback: Pulsing animation + waveform
- Speech-to-text appears in real-time
- User **releases** to stop recording
- Text can be edited afterward with keyboard

**Estimated time savings**:
- Typing "Completed homework assignment": ~5-8 seconds
- Speaking "Completed homework assignment": ~2 seconds
- **Savings**: 3-6 seconds per text input
